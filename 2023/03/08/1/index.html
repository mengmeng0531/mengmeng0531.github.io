Java 实现汉诺塔游戏的代码示例
Keep Team Lv2
  2023-03-08 16:36:24 2023-03-08 16:36   2023-03-08 16:47:19 

public class HanoiTower {

    public static void main(String[] args) {
        int n = 3; // 汉诺塔的层数
        char from = 'A'; // 源柱子的名称
        char to = 'C'; // 目标柱子的名称
        char temp = 'B'; // 辅助柱子的名称

        move(n, from, to, temp);
    }

    /**
     * 汉诺塔移动函数
     *
     * @param n     汉诺塔的层数
     * @param from  源柱子的名称
     * @param to    目标柱子的名称
     * @param temp  辅助柱子的名称
     */
    public static void move(int n, char from, char to, char temp) {
        if (n == 1) {
            System.out.println(from + " -> " + to);
            return;
        }

        move(n - 1, from, temp, to); // 将 n-1 个盘子从源柱子移动到辅助柱子
        System.out.println(from + " -> " + to); // 将最底下的盘子从源柱子移动到目标柱子
        move(n - 1, temp, to, from); // 将 n-1 个盘子从辅助柱子移动到目标柱子
    }
}

在此示例中，move() 函数是一个递归函数，它将从源柱子移动 n 个盘子到目标柱子，利用辅助柱子来协助移动。

该示例中默认情况下，汉诺塔的层数为 3，源柱子的名称为 ‘A’，目标柱子的名称为 ‘C’，辅助柱子的名称为 ‘B’。在调用 move() 函数时，传入这些参数即可开始游戏。

运行示例程序，输出结果如下：


A -> C
A -> B
C -> B
A -> C
B -> A
B -> C
A -> C

以上输出是每一步的移动过程。您可以通过修改示例程序中的参数来改变汉诺塔的层数和柱子名称，以便更好地理解游戏的原理。
